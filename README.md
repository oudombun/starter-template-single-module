Architecture starter template (single module)
==================

This template is compatible with the latest **stable** version of Android Studio.



## 1. 'main' Branch Features
* UI in Compose, list + write (Material3)
* Room Database
* Hilt
* ViewModel, read+write
* Navigation
* Repository and data source
* Kotlin Coroutines and Flow

## 2. 'xml' Branch Features
* UI in XML (XML layouts, ViewBinding, Fragment, Material Components)
* Room Database
* Hilt
* ViewModel, read+write
* Navigation
* Repository and data source
* Kotlin Coroutines and Flow
